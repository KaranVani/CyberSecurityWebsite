The Importance Of File Extensions

Published on April 5th, 2021 by Fin Watling

File extensions are the bit that comes after the file name, after the dot. They tell Windows (or whatever operating system you use) the type of file that it is. For example, .exe means executable, .zip means that it is a zipped or compressed file, .doc is a document file such as a microsoft word file and .png is a picture file.

Aside from wanting to know what files are on your computer, there are some really important things to keep in mind to make sure you are secure on your PC.

Executable files, or .exe files are files that when clicked, complete a series of instructions one after the other, this can be great if the file is an installer for something you want on your computer, however, if the file is malicious but not detected by your antivirus, this type of file can be used to easily infect your computer with a virus.
